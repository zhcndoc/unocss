# Zed 扩展

一个社区[扩展](https://github.com/bajrangCoder/zed-unocss)，为 Zed 提供对 UnoCSS 的支持。

:::info
注意：这是一个**社区维护的扩展，未由 UnoCSS 团队审核或维护**。请在使用前自行进行调研。如果您在使用该扩展时遇到任何问题，请向 [bajrangCoder/zed-unocss](https://github.com/bajrangCoder/zed-unocss) 反馈。
:::

## 安装

通过 Zed 扩展管理器安装 unocss 扩展，https://zed.dev/extensions/unocss

## Bug 报告 / 功能请求

请在 [问题追踪器](https://github.com/bajrangCoder/zed-unocss/issues) 中提交。
