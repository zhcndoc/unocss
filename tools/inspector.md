# 检查器

UnoCSS 的检查器 UI：`@unocss/inspector`。
随 `unocss` 和 `@unocss/vite` 一起提供。

## 使用方法

在您的 Vite 开发服务器中访问 localhost:5173/\_\_unocss 以查看检查器。

检查器允许您检查生成的 CSS 规则和每个文件应用的类。它还提供了一个 REPL，以便根据您当前的配置测试您的工具。
