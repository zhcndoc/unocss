包 ​

UnoCSS 是一个包含多个包的单体仓库。此页面列出了所有包及其在 unocss 包中的包含情况：

包描述包含在 unocss启用
@unocss/core没有预设的核心库✅-
@unocss/cliUnoCSS 的命令行接口✅-
@unocss/preset-mini最小但必要的规则和变体✅✅
@unocss/preset-wind3Tailwind CSS / Windi CSS 精简预设✅✅
@unocss/preset-wind4Tailwind4 CSS 精简预设✅✅
@unocss/preset-attributify启用 Attributify 模式以支持其他规则✅否
@unocss/preset-tagify启用 Tagify 模式以支持其他规则✅否
@unocss/preset-icons基于 Iconify 的纯 CSS 图标解决方案✅否
@unocss/preset-web-fonts支持网页字体（Google 字体等）✅否
@unocss/preset-typography排版预设✅否
@unocss/preset-rem-to-px将 rem 转换为 px 用于实用工具否否
@unocss/preset-legacy-compat用于旧版兼容性的工具集合否否
@unocss/transformer-variant-groupWindi CSS 的变体组功能的转换器✅否
@unocss/transformer-directives用于 CSS 指令如 @apply 的转换器✅否
@unocss/transformer-compile-class将一组类编译成一个类✅否
@unocss/transformer-attributify-jsx支持在 JSX/TSX 中使用无值的属性✅否
@unocss/extractor-pugPug 的提取器否-
@unocss/extractor-svelteSvelte 的提取器否-
@unocss/autocomplete自动补全工具否-
@unocss/config配置文件加载器✅-
@unocss/reset常见 CSS 重置的集合✅否
@unocss/viteVite 插件✅-
@unocss/inspectorUnoCSS 的检查器 UI✅-
@unocss/astroAstro 集成✅-
@unocss/webpackWebpack 插件否-
@unocss/nuxtNuxt 模块否-
@unocss/svelte-scopedSvelte Scoped Vite 插件 + 预处理器否-
@unocss/nextNext.js 插件否-
@unocss/runtimeUnoCSS 的 CSS-in-JS 运行时否-
@unocss/eslint-pluginESLint 插件否-
@unocss/eslint-configESLint 配置否-
